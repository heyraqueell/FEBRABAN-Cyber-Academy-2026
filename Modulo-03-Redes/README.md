# 📂 Módulo 03: Redes de Computadores e Segurança Cibernética

Este módulo está estruturado em três partes fundamentais para compreendermos a base da comunicação digital e como a segurança se aplica nesse ambiente.

### 📚 Conteúdo do Módulo
* **[Parte 01 - Fundamentos](./Parte-01/README.md):** Introdução às redes, componentes, topologias e métricas de desempenho.
* **[Parte 02 - Protocolos](./Parte-02/README.md):** Entendimento do modelo OSI/TCP/IP, encapsulamento e tipos de fluxo/difusão.
* **[Parte 03 - Segurança de Redes](./Parte-03/README.md):** Aplicação de defesas e ativos de segurança (em breve).

---

## Dica como estudar este conteúdo ✨

A área de redes possui muitos termos técnicos que podem parecer densos à primeira vista. Para facilitar seu aprendizado e retenção, recomendo fortemente o uso do **[NotebookLM](https://notebooklm.google.com/)**.

### Passo a passo para criar seu tutor de estudos pessoal:

1. **Copie o conteúdo:** Acesse cada uma das partes acima e copie todo o conteúdo (texto e explicações) do arquivo `README.md`.
2. **Crie um novo Notebook:** No [NotebookLM](https://notebooklm.google.com/), crie um "Novo Notebook".
3. **Adicione como Fonte:** Cole o conteúdo copiado na opção de "Fonte" (texto ou documento). Você pode repetir o processo para todas as partes do módulo (01, 02 e 03) para que a IA tenha uma base completa.
4. **Estude de forma ativa:** Com os dados carregados, utilize a plataforma para:
   - **Gerar testes:** Peça para a IA criar um quiz baseado nos termos técnicos.
   - **Criar resumos:** Solicite resumos em tópicos para revisão rápida.
   - **Vídeo autoexplicativo:** Utilize a ferramenta para gerar um guia de áudio (ou resumo guiado) sobre o tema.
   - **Tire dúvidas:** Pergunte ao chat: *"Como eu diferenciaria um Host de um Servidor de forma prática?"*.

**Dica Extra:** Utilize o NotebookLM para criar um "Mapa Mental" ou uma tabela comparativa dos protocolos. Isso ajuda muito a memorizar o que cada camada do modelo OSI faz!
