# 🛡️ FEBRABAN Cyber Academy 2026

Repositório destinado a resumos e estudos do curso Cyber Academy (FEBRABAN). Este espaço funciona como uma base de conhecimento ativa e portfólio técnico sobre Segurança da Informação e Cibersegurança.

### Objetivo do Repositório
Documentar o aprendizado teórico e prático adquirido ao longo das aulas, organizando os conceitos essenciais de segurança digital, governança, proteção de dados e gestão de riscos.

---

*   Modelos de Governança (Cubo de McCumber) e Normas Internacionais (ISO 27001 e ISO 27002).
*   Glossário Técnico Base e Introdução à Gestão de Riscos.
